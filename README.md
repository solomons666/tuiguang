## 自定义修改

### 修改主色调
在 `style.css` 中搜索 `#E60027` 并替换为你想要的颜色

### 修改字体
在 `style.css` 的 `body` 选择器中修改 `font-family`

### 调整动画速度
在 `style.css` 中搜索 `@keyframes` 并调整动画持续时间
